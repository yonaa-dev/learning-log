# Learning Log

이 저장소는 개발자로 일하며 느꼈던 한계와 고민을 바탕으로, 기초 개념부터 다시 정리하며 **학습 과정과 사고의 변화를 기록하기 위한 개인 학습 로그**입니다.

단순한 튜토리얼 정리가 아니라 **“왜 이 주제를 다시 학습했고, 무엇을 놓치고 있었는지”** 를 남기는 것을 목표로 합니다.

---

## Why this repository exists

이 저장소는 성장 중인 나 자신을 위한 기록이자, 개발자로서의 방향성과 정체성을 다시 정리하기 위해 기초 개념부터 설계 관점까지 차분히 되짚고 있습니다.

개발 경험이 충분히 쌓이지 못했다고 느낀만큼 **“어디까지 알고 있었는지”보다 “무엇을 이해하지 못한 채 지나왔는지”** 를 다시 점검하기 위한 기록입니다.

---

## What I document here

이 저장소에는 다음과 같은 내용을 기록합니다.

- 강의 및 학습 중 중요하다고 느낀 포인트
- 스터디 발표를 통해 설명했던 주제
- 이해가 어려웠던 부분과 그에 대한 나만의 해석
- 실무 또는 프로젝트에 어떻게 적용할 수 있을지에 대한 고민
- 개발 관련 행사, 북토크, 네트워킹 참여 후의 생각 정리

---

## Repository Structure

- `frontend/`
  - JavaScript, React 등 프론트엔드 핵심 개념 정리
  - 개념 → 헷갈린 지점 → 나만의 해석 순으로 정리
- `weekly/`
  - 매주 학습 내용과 회고
  - 이해한 것 / 막혔던 것 / 다음 주 계획 포함


---

## How I write

이 저장소는 완성된 정답 노트가 아니라,  
**생각이 정리되어 가는 과정을 기록하는 공간**입니다.

- 완벽히 이해한 뒤에만 정리하지 않는다
- 헷갈렸던 지점과 고민을 그대로 남긴다
- 코드보다 **“왜 이렇게 생각했는지”** 를 더 중요하게 기록한다
- 나중에 면접에서 말로 설명할 수 있는 수준을 목표로 한다

---

## Goal

이 저장소의 목표는 단순한 지식 축적이 아니라, 문제를 어떻게 이해하고, 어떤 기준으로 판단했는지를 설명할 수 있는 개발자로 성장하는 것입니다.

각 기록은
- 개념을 어떻게 재정의했는지
- 실무 또는 프로젝트에 어떻게 연결할 수 있는지
- 이후 어떤 질문으로 확장되었는지

를 남기는 데 초점을 둡니다.  
다시 기초부터 쌓아가는 과정을 **스스로 증명하기 위한 학습 기록**이기도 합니다.
